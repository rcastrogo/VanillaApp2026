2. El Servicio I18n
Necesitas un objeto global o un "Helper" que gestione el idioma actual y realice la sustitución de variables.

TypeScript
class I18n {
  private currentLang = 'es';

  setLanguage(lang: string) {
    this.currentLang = lang;
    // Aquí podrías disparar un evento global para que todos los componentes se actualicen
    window.dispatchEvent(new CustomEvent('langChange'));
  }

  // El método "t" (translate)
  t(key: string, params: Record<string, any> = {}) {
    let text = translations[this.currentLang][key] || key;

    // Interpolación simple: reemplaza {name} por el valor en params
    Object.keys(params).forEach(param => {
      text = text.replace(`{${param}}`, params[param]);
    });

    return text;
  }
}

export const i18n = new I18n();
3. Integración con tu buildAndInterpolate
Para que no tengas que llamar a i18n.t() manualmente en cada componente, puedes crear una directiva de traducción o una sintaxis especial en tus templates.

Opción A: Directiva data-t
Puedes añadir una hidratación que busque etiquetas con traducción:

HTML
<button data-t="logout"></button>
<p data-t="welcome" data-t-params='{"name": "Gabi"}'></p>
En tu lógica de hidratación:

TypeScript
function hydrateI18n(el: HTMLElement) {
  const keys = el.querySelectorAll('[data-t]');
  keys.forEach(node => {
    const key = node.getAttribute('data-t');
    const params = JSON.parse(node.getAttribute('data-t-params') || '{}');
    node.textContent = i18n.t(key, params);
  });
}
Opción B: Interpolación Directa {{t:key}}
Si tu sistema de strings ya detecta llaves, puedes añadir un prefijo:

TypeScript
// Dentro de tu función de interpolar strings
function interpolateString(template: string, ctx: any) {
  return template.replace(/{{t:(.*?)}}/g, (_, key) => i18n.t(key.trim()));
}
4. ¿Cómo manejar el cambio de idioma sin recargar?
Dado que tienes un método update() en tus componentes, puedes hacer que todos los componentes activos se re-rendericen cuando el idioma cambie.

TypeScript
// En tu BaseComponent
constructor(el: HTMLElement) {
  // ... tu lógica de props y children ...

  window.addEventListener('langChange', () => {
    this.update(); // Forzamos el re-renderizado con el nuevo idioma
  });
}
